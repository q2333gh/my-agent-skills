## my-agent-skills

我自用的 skills / SOP 笔记仓库：把坑整理成文本，交给其他 LLM agent 先读一遍，执行时就不容易翻车。

- `install-win-ssh.md`: 在 Win11 上安装 / 启用 OpenSSH Server 的步骤（含内置功能 + winget 备用方案），专门给 Agent 用。
- `ssh-to-win.md`: Termux/WSL → Win11 的 SSH 公钥免密登录避坑（管理员账号 / `administrators_authorized_keys` / ACL / UAC）。

后续可以在本仓库按文件一个技能的方式，持续补充更多「专门给 Agent 看的操作攻略」。
